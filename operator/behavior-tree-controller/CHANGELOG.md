# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-11-04

### Added
- Initial release of behavior tree controller extension
- Graphical control panel for sending behavior commands to state machine
- Support for all behavior commands:
  - Arm/Disarm commands
  - Auto Takeoff/Land commands
  - Emergency Stop command
  - Survey command
  - Geofence Mapping command
  - Search command
- Configurable topic path with robot namespace support
- Status feedback for command sending
- Chinese language interface
