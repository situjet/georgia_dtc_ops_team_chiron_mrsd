# Geofence & Human GPS Visualizer

Foxglove Studio 扩展，用于在交互式地图上可视化地理围栏边界和检测到的人员 GPS 坐标。

## 功能

- 可视化地理围栏边界
- 显示检测到的人员 GPS 位置
- 交互式地图界面

## 安装

```bash
npm install
npm run build
npm run local-install
```

## 使用

1. 在 Foxglove Studio 中添加 "Geofence & Human GPS Visualizer" 面板
2. 连接到 ROS2 系统
3. 面板将自动订阅相关话题并显示数据
